# Campanha-Hub
Este CampanhaHub é uma plataforma completa para gestão de campanhas políticas, focada em eficiência e organização. O sistema oferece ferramentas para cadastro de eleitores, agenda de eventos, controle de tarefas e analytics em tempo real. Ideal para candidatos que buscam centralizar dados e otimizar o engajamento com sua base de apoiadores totais.
